# Mini CRM + Automatización (Portafolio)

App web en HTML/CSS/JS que simula un flujo de ventas: registro de leads, pipeline y recordatorios automáticos.

## Funcionalidades
- Registro de leads (nombre, WhatsApp, interés)
- Pipeline: Nuevo → Contactado → Calificado → Cerrado / Perdido
- Automatización (simulada): si pasan 2 días sin actividad en Nuevo/Contactado, marca “Requiere seguimiento”
- Persistencia con LocalStorage
- Resumen de métricas por etapa

## Tecnologías
- HTML, CSS, JavaScript (Vanilla)
- LocalStorage

## Cómo ejecutarlo
1. Descarga o clona el repositorio
2. Abre `index.html` en tu navegador
